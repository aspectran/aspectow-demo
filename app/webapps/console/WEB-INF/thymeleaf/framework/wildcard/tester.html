Token Reference
*
Matches 0 or more characters within a segment (cannot cross separators).
**
Matches 0 or more characters across multiple segments (includes separators).
?
Matches 0 or 1 optional character within a segment.
+
Matches exactly 1 character within a segment (cannot be empty).
